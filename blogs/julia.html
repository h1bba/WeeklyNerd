Animaties met karakter

Julia Miocene

Julia Miocene is een digitale kunstenaar, animator, modeler en noem het maar op. Cassie Evans en Julia hebben echter een andere style. Julia gebruikt geen SVG's en animeert ze ook niet, ook gebruikt ze geen GSAP voor haar animaties.

Julia gebruikt pure CSS en HTML om "web" kunst te maken. De enige HTML element die ze gebruikt in haar werk is 'div', verder gebruikt ze geen enkel ander element. Haar style is puur gebaseerd op visuele web kunst en is daarom minder bezig met semantische elementen.

Julia maakt gebruik van pseudo classes zoals ::before en ::after. Ook gebruikt ze veel CSS properties om te zorgen dat de styling goed is. Dingen zoals position: absolute, transform, clip-path, border-radius, transform-origin etc. Met het gebruik van deze styling properties kan ze in principe elke vorm maken die ze nodig kan hebben.

Stel je voor Julia wilt een ontwerp maken met een mens. Deze mens die deconstrueert ze in haar hoofd en deelt ze op in kleine deeltjes. Een menselijk hand bestaat bijvoorbeeld uit 5 vingers, maar Julia die zorgt er nog voor dat elke vinger ook een bot bevat, waardoor de vinger ook buigend te animeren is. Door juiste gebruik van position: relative, kan ze meerdere vormen bij elkaar groeperen en animeren.

Julia is niet bang om te experimenteren en voegt vaak nog een extra interactie toe waardoor de kunst ook nog eens interactief is. Ze zou wel gek moeten worden van alle 'div' gebruik en het terugvinden van elementen omdat het er zo ontzettend veel kunnen zijn.
