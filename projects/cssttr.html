CSS To The Rescue

Voor het vak CSSTTR heb ik er voor gekozen om als concept een character customization scherm te maken. Dit vak heefte strakke richtlijnen waar we met (nieuwe) CSS selectors een interactief product gaan maken. Het enige stukje JavaScript wat we mogen gebruiken is voor een slider.

Wat er goed ging bij dit vak was de logica ontleden, ik had Sanne 't Hooft zijn eigen project bekeken als inspiratie en keek naar de code. Het gebruik van pseudo classes was interessant, stel je voor je hebt 2 wenkbrauwen, dit hoeft niet persee 2 elementen op zichzelf te worden in HTML, want we kunnen ::before en ::after koppelen op bijvoorbeeld de ogen etc.

Wat ik vooral heb verbeterd is het gebruik van selectors zoals :has(), :checked, en een beter begrip voor de selectors zelf, ook heb geleerd dat CSS nesting een ding is wat ik voorheen niet wist. Ik heb ook geleerd hoe ik variabeles kan gebruiken om dynamisch waardes door te geven, bijvoorbeeld een feedforward effect. Clip-path heb ik ook beter leren gebruiken en zelfs geleerd hoe ik het kan animeren.

Wat er bij dit vak moeilijk was, was het concept verzinnen. Door de voorwaardes moest ik goed nadenken over wat voor interactief product ik nou echt kan maken zonder het gebruik van JavaScript. Wat ik ook beter kon doen was het beter gebruiken van positioning, omdat ik veel position absolute's heb gebruikts waardoor het responsive maken, of animeren een beetje een challenge werd.
